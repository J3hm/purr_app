# F0 Purring cat 🐾

A soothing, organic purr machine for the Flipper Zero.

## Features
- **Organic Vibration:** Not a flat rhythm, but a living breath with micro-variations.
- **Random Trills:** Press **OK** to hear 3 different types of cat sounds (Question, Answer, or a long Sigh).
- **Safe Power:** Vibrations are calibrated to be soothing, not noisy for the Flipper's hardware.
- **Custom Icon:** Features a unique cat paw icon designed by J3hm.

## Controls
- **UP / DOWN:** Adjust purr intensity (Power).
- **LEFT / RIGHT:** Adjust purr speed (Rhythm).
- **OK (Short):** Meow / Trill.
- **OK (Long):** Stop / Start.
- **BACK:** Exit.
